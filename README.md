POO TADS 2016.1
===============

Código fonte da disciplina de POO IFRN 2016.1

Dependências
------------

* org.reflections: 0.9.9-RC2
* junit: 3.8.1
* maven: 3.3.9

How To
------

```bash
# build
mvn clean install
# execução
mvn exec:java -Dexec.mainClass=br.eng.augusteiner.poo.<NomeDaClasseAqui> -pl <NomeDoSubProjetoAqui>
```
